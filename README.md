# EvaluacionParcial_MiPerfilProfesional
## Mi Perfil Profesional
**Mi Perfil Profesional** La aplicación Mi Perfil Profesional es una app sencilla  permite visualizar la imagen de perfil, el nombre,apellido  y la carrera profesional de una persona. Además, se puede ingresar una descripción personal mediante un campo de texto y utilizar los botones de “Guardar” o “Editar” para administrar la información registrada.
